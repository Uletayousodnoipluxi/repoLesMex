Система онлайн-бронирование отелей с поиском и управлением бронированиями.
##Основные функции
* Поиск отелей по датам, местоположению и количеству гостей
* Просмотр деталей отеля и доступных номеров
* Процесс бронирования с подтверждением
* Личный кабинет пользователя с историей бронирований
## Быстрый старт
1. Клонируйте репозиторий:
\'\'\'bash
git clone https://github.com/your-username/hotel-booking-system.git\'\'\'
2. Установите зависимости:
\'\'\'bash

\'\'\'
3. Запустите в Docker:
\'\'\'bash

\'\'\'
